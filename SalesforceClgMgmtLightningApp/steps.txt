🔷 3. Validation Rules
✅ 3.1 Student Validation Rules
🔹 1. Roll Number Validation
Rule Name: Roll_Number_Validation
Description: Roll number must be greater than 0

Formula:

Roll_Number__c <= 0

Error Message:
Roll Number must be greater than 0

🔹 2. Email Validation
Rule Name: Email_Format_Validation
Description: Email must contain @ symbol

Formula:

NOT(CONTAINS(Email__c__c, "@"))

Error Message:
Email must contain @ symbol

🔹 3. Student Name Validation
Rule Name: Student_Name_Validation
Description: Student name cannot be empty

Formula:

ISBLANK(Student_Name__c)

Error Message:
Student Name cannot be empty

🔹 4. Marks Validation
Rule Name: Marks_Validation
Description: Marks must be between 0 and 100

Formula:

OR(Marks__c < 0, Marks__c > 100)

Error Message:
Marks must be between 0 and 100

🔹 5. Department Validation
Rule Name: Department_Validation
Description: Department must be selected

Formula:

ISBLANK(TEXT(Department__c))

Error Message:
Department must be selected

✅ 3.2 Faculty Validation Rules
🔹 1. Faculty Name Validation
Rule Name: Faculty_Name_Validation
Description: Faculty name must have at least 3 characters

Formula:

LEN(Faculty_Name__c) < 3

Error Message:
Faculty Name must contain at least 3 characters

🔹 2. Faculty ID Validation
Rule Name: Faculty_ID_Validation
Description: Faculty ID must be greater than 0

Formula:

Faculty_ID__c <= 0

Error Message:
Faculty ID must be greater than 0

🔹 3. Salary Validation
Rule Name: Salary_Validation
Description: Salary must be within valid range

Formula:

OR(Salary__c <= 10000, Salary__c >= 500000)

Error Message:
Salary must be between 10,000 and 500,000

🔹 4. Joining Date Validation
Rule Name: Joining_Date_Validation
Description: Joining date cannot be future date

Formula:

Joining_Date__c > TODAY()

Error Message:
Joining date cannot be a future date
