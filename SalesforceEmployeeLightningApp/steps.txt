Object Manager → Employee → Validation Rules → New

VALIDATION 1 — Employee Name
Rule Name
Employee_Name_Validation
Formula
OR(
    ISBLANK(Employee_Name__c),
    LEN(Employee_Name__c) < 3
)
Error Message
Employee Name must contain at least 3 characters.

Save.

VALIDATION 2 — Employee ID
Rule Name
Employee_ID_Validation
Formula
Employee_ID__c <= 0
Error Message
Employee ID must be greater than 0.

Save.

VALIDATION 3 — Salary Validation
Rule Name
Salary_Validation
Formula
OR(
    Salary__c <= 10000,
    Salary__c >= 500000
)
Error Message
Salary must be between 10,000 and 500,000.

Save.

VALIDATION 4 — Department Required
Rule Name
Department_Validation
Formula
ISBLANK(TEXT(Department__c))
Error Message
Please select a department.

Save.

VALIDATION 5 — Joining Date Validation
Rule Name
Joining_Date_Validation
Formula
Joining_Date__c > TODAY()
Error Message
Joining Date cannot be a future date.

Save.
